プログラムを読んでみる

プログラムの流れ

先ほどのプログラムを読んでいきます。
各処理の細かいことは追々解説しますので、ここでは簡単に説明します。

さてプログラムを読み解くために意識することがあります。
それは「プログラムは基本的に上から下に流れる。」というものです。
これはそのままの意味でプログラムは上から下に処理されていきます。

これを前提に読み解いていきます。


C言語における関数

C言語は、小さい機能ごとに分割してプログラムを記述します。
例えば...
  • 画面に表示する処理
  • プレイヤーを操作する処理
  • ダメージを計算する処理
  • 計算したダメージをHPから減らす処理
といった感じで、処理を分割して、仕様通りの順番に呼び出していきます。
この分割した状態のプログラムを「関数」と呼んでいます。


今回のプログラムは次の2つが関数です。
main関数
/* メイン関数 */
int main()
{
    int sum;

    /* 1から10まで加算する */
    sum = add(1, 10);

    printf("計算結果：%d\n", sum);

    return 0;
}
add関数
/* iからjまで加算して結果を戻す関数 */
int add(int i, int j)
{
    int n, sum = 0;

    for (n = i; n <= j; n++)
    {
        sum += n;
    }

    return sum;
}


メイン関数から始まる

C言語は「必ずメイン関数から開始」されます。
この関数ですね。
/* メイン関数 */
int main()
{
    int sum;

    /* 1から10まで加算する */
    sum = add(1, 10);

    printf("計算結果：%d\n", sum);

    return 0;
}
この最初に実行される関数のことを、メイン関数とは別に
「エントリーポイント」とも呼ぶことがあります。

関数の中身は「{」から「}」の間に記述されます。
これを「ブロック」と呼びます。


なので、上のプログラムの場合、一番最初に実行される処理は
int sum;
になります。
