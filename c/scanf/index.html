文字や数値を入力する(scanf関数)

scanf関数とは

アプリケーションに対して、キーボードなどから文字や数値を入力し、
入力されたデータを変数に代入してくれる関数です。

サンプルとして、次のプログラムを入力し実行までしてみてください。
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int a;

    printf("何か数値を入力：");
    scanf("%d", &a);

    printf("入力された数値は「%d」ですね。\n", a);

    return 0;
}
実行すると『何か数値を入力：』と表示されて止まるので、キーボードから
数値を入力してEnterキーを押してください。
そうすると『入力された数値は「○○」ですね。』と表示されます。
